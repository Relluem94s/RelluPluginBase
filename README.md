A Base Plugin based on RelluEssentials

# How to use

## Development / Build
1. ```shell
   git clone https://github.com/Relluem94/RelluEssentials.git
   ```

1. ```shell
   git checkout $(git describe --tags $(git rev-list --tags --max-count=1))
   ```

1. ```shell
   mvn clean install
   ```
1. Fork this Repository 
1. Edit POM
1. Create your own stuff
1. Build it!
